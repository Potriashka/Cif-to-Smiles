# Cif-to-Smiles
An app that will allow your .cif file to be easily converted to a string of SMILES 

You have to put the converter script and your .cif file in the same directory, and the output will be SMILES for the compound in .cif file.

# bad.py
A bad version which partially works. You can find one of my .cif files in the [.cif] directory in the repository and try it out. It will get an almost correct output, although double bonds will be missing. It also may not work on your compound, you may try it out and send feedback.
